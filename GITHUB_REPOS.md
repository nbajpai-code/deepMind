# DeepMind GitHub Repositories

Key open-source projects hosted by `google-deepmind`.

## 1. Research Tools & Libraries

### [OpenSpiel](https://github.com/google-deepmind/open_spiel)
*   **Description:** A collection of environments and algorithms for research in general reinforcement learning and search/planning in games.
*   **Use Case:** Game theory, Multi-agent RL.

### [Sonnet](https://github.com/google-deepmind/sonnet)
*   **Description:** TensorFlow-based neural network library.
*   **Use Case:** Building complex neural network architectures.

### [TRFL](https://github.com/google-deepmind/trfl)
*   **Description:** TensorFlow Reinforcement Learning. A library of RL building blocks.
*   **Use Case:** Implementing RL agents.

### [Graph Nets](https://github.com/google-deepmind/graph_nets)
*   **Description:** Build Graph Neural Networks in TensorFlow.
*   **Use Case:** Graph-structured data learning.

### [Lab](https://github.com/google-deepmind/lab)
*   **Description:** A 3D learning environment based on the Quake III Arena engine.
*   **Use Case:** Agent training in 3D worlds.

## 2. Environments

### [dm_control](https://github.com/google-deepmind/dm_control)
*   **Description:** DeepMind Control Suite. Python-based physics simulation engine (MuJoCo).
*   **Use Case:** Developing and testing RL algorithms for continuous control.

### [Acme](https://github.com/google-deepmind/acme)
*   **Description:** A library of reinforcement learning agents and agent building blocks.
*   **Use Case:** Rapid prototyping of RL agents.
